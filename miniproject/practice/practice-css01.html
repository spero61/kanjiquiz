style.css


    body {
      background-color: rgb(1, 7, 24);
      color: aliceblue;
    }

    h1 {
      font-size: 50px;
    }
  1. 이번에는 지난 시간에 만든 raami.html 를 활용하여
    css 실습을 진행하도록 하겠습니다.
    이번에도 raami.html이 포함된 폴더 채로 VSCode에서
    실행하여 live server로 css적용시킬 때마다의 변화를
    한 줄씩 직접 눈으로 확인해보며 실습을 진행하기 바랍니다.
  2. 먼저 루트(root) 폴더에 css 라는 이름의 폴더를 생성하고
    그 안에 style.css 파일을 생성합니다.
    (여기서 root란 웹페이지의 최상위 디렉토리로서
    index.html 이 위치하는 디렉토리를 의미합니다)
  3. style.css 파일 안에 위에 코드 내용을
    (body의 배경색과 body 안의 문자 색상을 지정하는 css 코드,
    그리고 h1 태그의 폰트 사이즈를 50px로 지정하는 css 코드)
    새로 만든 style.css 파일 안에 붙여 넣습니다.
  4. 이제 html에 css 파일을 연결해주는 link 태그를 통해
    raami.html 에 style.css 파일 안의 css 코드 내용을 적용해 봅시다
    아래 코드를 head 태그 안에 넣어줍시다

    <link rel="stylesheet" href="./css/style.css">
  5. 단 몇 줄의 css 코드만으로도 분위기가 변했습니다!
    여기에 이어 학습한 내용을 바탕으로 자유롭게 실습을 진행해봅시다.
  6. 어느 정도 실습을 마쳤으면 아래의 샘플 페이지와
    샘플 페이지에 쓰인 css 코드(링크 아래 부분)를 보면서
    혹시 사용하지 않은 css 속성 등을 조금 더 연습해보세요
  7. css 적용 샘플 페이지 ← 클릭
  8. 다음으로 웹폰트를 지정하는 방법을 익혀봅시다
    유저가 사용하고 있는 운영체제, 설치된 시스템 폰트 등은 서로 다르므로
    개발자 / 디자이너가 원하는 폰트를 손쉽게 웹페이지에 적용하기 위해서는
    웹 폰트(웹페이지가 로드될 때 폰트를 자동으로 다운받도록 함)를
    이용하는 방법이 있습니다.
    가장 손쉽게 이용가능한 Google Fonts 서비스를 이용해서
    실습에 적용해봅시다!
  9. 구글 검색창에 구글 폰트(google fonts)로 검색하거나 아래 링크를 이용해서
    구글 폰트 서비스 웹페이지로 이동합니다.
  10. 구글 폰트 페이지 ← 클릭
  11. 웹페이지 상단 검색 창 바로 아래에 언어를 고를 수 있는
    Language 를 이용해서 원하는 언어를 지원하는 폰트를 쉽게
    지정하여 볼 수 있습니다.
    (Categories로 Serif 체, Sans Serif 체, 손글씨 체 등
    폰트 서체에 따라 세분화된 검색 결과를 볼 수도 있습니다.)
    마음에 드는 폰트를 찾았다면 클릭해줍시다
  12. 서로 다른 font-weight 에 따른 폰트 예시가 보입니다.
    원하는 굵기의 폰트를 찾았다면
    오른쪽에 있는 + Select this style 버튼을 눌러줍시다
  13. 우측에 선택한 폰트가 담겼습니다
    현재는 Review 탭이 선택되어 있는데 그 오른쪽에 있는
    Embed 탭을 클릭해줍시다
  14. 코드가 두 가지 나옵니다.
    상단은 html의 head 태그 안에 들어가는 link 태그이고
    하단의 코드는 css 파일 안에 적용시키기 위한 속성과 속성값을 의미합니다.
    자 이제, 선택한 폰트를 raami.html 페이지에 적용해봅시다!
  15. 여유가 있을 때 심심풀이로 다양한 css 속성과 속성값을
    새로운 html 페이지로 만들어서 연습해보고
    심플하지만 잘 디자인된 웹페이지의 css 코드를 보고
    하나하나 분석해보면서 css 스킬을 습득하는 방법을 추천합니다.
  16. ご苦労さん!
