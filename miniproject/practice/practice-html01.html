    먼저 raami.html 을 웹 브라우저에서 실행시키거나,
    VSCode의 live server 기능을 이용해여 웹 브라우저에 표시해봅시다.

  3. 텍스트 편집기로 이동해서 body태그 안에 있는
    h1 태그(내용 : Raami)를 찾아 바로 아래에
    div 태그를 만들고 div 태그의 id는 intro로 지정해줍니다
  4. div 태그 안에 h2 태그를 이용해서 아래 텍스트를 그 안에 내용으로 입력
    Raami – a new dining collection by Jasper Morrison
  5. p 태그를 만들고 안에 내용을 아래와 같이 입력

    라미(Raami)는 재스퍼 모리슨(Jasper Morrison)이 디자인한 이딸라의 새로운 다이닝 컬렉션입니다. <br> 핀란드어로는 ‘프레임’을 의미하는 라미 컬렉션은, 제품이 그 공간의 분위기를 만들어내는 역할에 중점을 두었습니다. <br> 도자기, 유리 그리고 나무로 구성된 컬렉션으로 어떤 장소나 시간에 상관 없이 멋스러운 테이블 분위기를 만들어냅니다.
  6. div id="intro" 아래에 div id="main" 태그를 새로 만들기

    참고로 이 문단 이하에서 #main은 태그의 id 값이 main을 의미하며
    .content 라고 함은 태그의 class 값이 content를 의미합니다
    (css에서 # 는 id 선택자를 의미하고 . 은 class 선택자를 의미)
  7. div #main 안에는 h3 태그를 이용해서 '제품 종류' 라고 내용을 넣고
    그 아래에 ul태그를 이용해서 순서 없는 리스트를 만들어봅시다

    ul의 하위 태그로 li 태그(list 태그)가 있으며 리스트는 4개를 만들고 각각
    '텀블러', '카라페', '딥 플레이트', '캔들 홀더'라고 내용을 넣어줍니다
  8. ul태그 아래에는 h3 태그를 이용해서 '디자이너 인터뷰' 텍스트를 입력합니다
  9. h3태그 아래에 p 태그를 이용해서 아래 내용을 복붙해줍니다.

    "Ninety-nine per cent of all design objects are intended to be noticed quickly, that's part of the marketing strategy of a great many companies. But objects that you notice immediately are not usually the best everyday performers, they disturb the atmosphere by providing too much presence. Objects that contribute the best atmosphere tend to be less immediately noticeable; it may take some time before you appreciate them for their practicality and more subtle, discreet presence." Extract from Iittala Journal - No2 - September 2018, interview with Jasper Morrison conducted by Marco Velardi.
