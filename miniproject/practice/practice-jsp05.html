  1. JSP 태그에는
    (1) JSP 커스텀 태그와 (2) JSTL(JSP Standard Tag Library)
    이렇게 두 가지가 있습니다.
    JSTL은 오라클에서 표준 태그 라이브러리를 지정하여 설정해 둔 것으로
    코어, 함수, 국제화, XML, 데이터베이스의 5가지가 있습니다
    이 중에서 이번 페이지에서는 코어(core) 태그와 함수(function) 태그를
    예제 파일을 통해 몇 가지 알아보도록 하겠습니다.
  2. JSTL을 이용하기 위해서는 JSTL 라이브러리가 필요합니다
    WebContent/WEB-INF/lib폴더에
    아래 링크를 통해 다운 받은 .jar 파일 2개를 붙여 넣어줍니다
  3. jstl-1.2.1 다운로드
  4. 여기 페이지 상단에 test.jsp 파일의 코드가 있습니다
    가장 JSTL의 core tag 중 set 을 하나 이용하여 현재 서버 시각을
    출력하는 가장 간단한 JSTL 예제입니다.
  5. 여기서 처음 등장하는 것이 다음 taglib 디렉티브 입니다 
    <%@ taglib uri="http://java.sun.com/jsp/jstl/core" prefix="c" %>
    여기에 있는 http://java.sun.com/jsp/jstl/core는
    uri로서 이 taglib 디렉티브를 포함한 jsp페이지를
    로컬 컴퓨터에서 tomcat 서버를 이용하여
    실행할 때 실제 인터넷 연결이 일어나지는 않습니다.
    다음으로 prefix="c" 에서 c는 core 태그를 의미합니다
    (함수태그는 prefix 값으로 "fn"을 이용)
